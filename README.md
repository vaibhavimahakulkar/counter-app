# 🔢 Counter App

A simple and interactive Counter App built using **HTML**, **CSS**, and **Vanilla JavaScript** to practice **DOM Manipulation** and **Event Handling**.

## ✨ Features

* ➕ Increase counter
* ➖ Decrease counter
* 🔄 Reset counter
* ⚡ Instant UI updates using DOM manipulation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 Concepts Practiced

* DOM Selection (`querySelector`)
* Event Listeners
* Functions
* Arrow Functions
* Variables
* Updating the DOM dynamically
* Separation of Logic and UI
* ES6 Syntax

## 🎯 Learning Outcome

This project helped me understand how JavaScript interacts with HTML using the DOM and how user actions (click events) can dynamically update a webpage. It also strengthened my understanding of organizing code by separating application logic from UI updates.
